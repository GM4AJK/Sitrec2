Three.js r180 - Local Copy
===========================

This directory contains just the essential Three.js files needed for the tools:

- three.core.js (1.3MB) - Core Three.js library
- three.module.js (589KB) - Module wrapper (imports from three.core.js)
- addons/controls/OrbitControls.js (38KB) - Orbit camera controls

Downloaded from: https://cdn.jsdelivr.net/npm/three@0.180.0/

This minimal installation provides the simplicity of a single-file setup
while eliminating runtime CDN dependencies.
